**Customer Personality Data Cleaning
Data cleaning and preparation project using Python (Pandas)**

**Overview**
This project focuses on cleaning and preprocessing a raw dataset using Python (Pandas). The goal was to prepare the data for analysis by handling missing values, fixing inconsistencies, and standardising formats.

**Tasks Performed**
- Identified and handled missing values in the Income column
- Checked and removed duplicate rows
- Standardised categorical data (e.g., marital status)
- Converted date column to datetime format
- Renamed columns for consistency (lowercase, no spaces)
- Verified and corrected data types

**Tools Used**
- Python
- Pandas

**Output**
- Cleaned dataset ready for analysis
